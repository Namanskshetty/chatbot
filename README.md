# chatbot
To run chat bot in your pc
### Prerequisite
* Python3

### Instalation
1. Clone the repo
   ```sh
   git clone https://github.com/Namanskshetty/chatbot.git
   ```
2. Install chatterbot
  ```sh
  pip install chatterbot==1.0.2
  ```
3. Install pytz 
  ```sh
   pip install pytz
  ```
4. Install Flask
    ```sh
    pip install flask
    ```
5. Install wikipedia
    ```sh
    pip install wikipedia
    ````
6. Install pyjokes
  ```sh 
  pip install pyjokes
  ```
###Run the code bash
1. Executable bash
   ```sh
   chmod +x bash.sh
   ```
2. Run bash 
   ```sh 
   ./bash.sh
   ```

To add your questions and answers for the chatbot. Use the file data.yml
